## 1. 🎯 Problema

A compra de ingressos para jogos de futebol locais ainda é, muitas vezes, um processo inconveniente e pouco acessível. Torcedores precisam se deslocar até pontos físicos de venda, enfrentar filas e lidar com horários restritos.

Esse cenário gera frustração, perda de tempo e, em muitos casos, desestimula a participação nos eventos, impactando diretamente a experiência do torcedor e a arrecadação dos organizadores.

---

## 2. 👥 Usuários

- Torcedores locais: pessoas que desejam adquirir ingressos de forma rápida, principalmente via celular.
- Organizadores: responsáveis pela gestão dos jogos e controle de acesso dos participantes.

---

## 3. 🧠 Solução

A solução proposta é um sistema web que digitaliza a compra de ingressos para jogos de futebol locais, oferecendo uma experiência prática, rápida e acessível aos torcedores. Por meio de uma interface intuitiva, os usuários podem visualizar partidas disponíveis, selecionar ingressos e concluir a compra diretamente pelo celular ou computador, eliminando a necessidade de deslocamentos e filas em pontos físicos.

Após a compra, o ingresso é gerado digitalmente com um QR Code único, que pode ser apresentado na entrada do evento para validação. Para os organizadores, o sistema disponibiliza um painel de controle que permite gerenciar jogos e validar ingressos em tempo real, garantindo mais eficiência operacional e segurança no acesso aos eventos.

---

## 4. 🔄 User Flow

```text
Home → Escolher jogo → Selecionar ingresso → Pagamento → QR Code
```
